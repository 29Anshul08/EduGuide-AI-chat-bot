# 🎓 EduGuide AI — Coaching Institute Chatbot

A modern, AI-powered chatbot assistant designed for coaching institutes. It helps students and parents get instant answers about courses, admissions, fees, batch timings, scholarships, and results — available 24/7.

## ✨ Features

- 💬 **Smart Chatbot** — Intent-based NLP engine with 15+ intents covering JEE, NEET, Foundation, Boards & CUET
- 🎨 **Premium UI** — Clean white theme with glassmorphism, smooth animations & responsive design
- 📋 **Lead Capture** — Built-in contact form for counselor callbacks & lead generation
- ⚡ **Quick Replies** — One-tap buttons for common queries like Courses, Fees, Timings, Results
- 📱 **Fully Responsive** — Seamless experience across desktop, tablet & mobile
- 🚀 **Zero Dependencies** — Pure HTML, CSS & vanilla JavaScript — lightweight & blazing fast
- 🎯 **Personalized Guidance** — Class-based course recommendations for students
- 🏆 **Results Showcase** — Highlights top rankers and selection statistics

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v14 or higher)

### Installation

```bash
# Clone the repository
git clone https://github.com/29Anshul08/EduGuide-AI-chat-bot.git

# Navigate to the project
cd EduGuide-AI-chat-bot

# Install dependencies
npm install

# Start the dev server
npm run dev
```

Open **http://localhost:3000** in your browser.

## 💬 Chatbot Capabilities

The chatbot understands and responds to queries about:
